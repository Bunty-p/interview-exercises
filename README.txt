To do these exercises fire up IDEA

File -> New Project
Create Java project from existing sources
Name: simplehashmap or friendlyurl (or other)
Project files location: vm-interview-exercises/friendlyurl (or other)
Accept all the default

It should drop you into the project and then you can open the src and test files. Go into the unit test file and right click 
and "Run" the unit test.

See README.txt in each project for suggestions/ideas
